# RPA Expense Tracker

## 📌 Description
This project automates expense tracking using UiPath. It reads expense data from Excel, processes it, generates summary reports, and sends automated email notifications with the report.

## ⚙️ Features
- Reads expense data from Excel
- Categorizes and processes expenses
- Generates summary reports
- Sends automated email with formatted summary
- Attaches Excel report in email

## 🛠️ Tech Used
- UiPath (RPA Tool)
- Excel
- Email Automation using UiPath Mail Activities

## 📁 Project Structure
- AutomatedExpenseTracker → Workflows (.xaml)
- Data → Input Excel file
- Reports → Generated output reports

## ▶️ How to Run
1. Open project in UiPath Studio
2. Run `Main.xaml`
3. Ensure Excel file is present in Data folder

## 🚀 Outcome
Automated end-to-end expense tracking and reporting process, reducing manual effort and improving efficiency.
